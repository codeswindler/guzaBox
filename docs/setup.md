# JazaBox Setup

## Backend Environment

Set these variables in your host environment or a local `.env` file:

- `PORT=4000`
- `DB_HOST=localhost`
- `DB_PORT=3306`
- `DB_USER=guzabox`
- `DB_PASSWORD=pass`
- `DB_NAME=jazabox`
- `DB_SYNC=true`
- `DB_LOGGING=false`
- `JWT_SECRET=change-me`
- `ADMIN_PHONE=254700000000`
- `ADMIN_EMAIL=admin@example.com`
- `SMS_PROVIDER=stub`
- `MPESA_BASE_URL=`
- `MPESA_SHORTCODE=`
- `MPESA_PASSKEY=`
- `MPESA_ACCESS_TOKEN=`
- `MPESA_CALLBACK_URL=`

## Dashboard Environment

Set:

- `NEXT_PUBLIC_API_URL=http://localhost:4000`
